# Desafio QA - Beedoo 2026

Este repositório contém a análise, cenários de teste, casos de teste e registro de bugs encontrados durante a avaliação da aplicação de cadastro e listagem de cursos.

## Objetivo da aplicação

A aplicação tem como objetivo permitir o cadastro e a visualização de cursos em uma plataforma simples de gerenciamento. O sistema possibilita registrar novos cursos e exibi-los em uma lista para consulta dos usuários.

## Principais fluxos disponíveis

1. Cadastro de cursos
O usuário pode preencher um formulário com as informações do curso e salvá-lo no sistema.

2. Listagem de cursos
Após o cadastro, os cursos devem ser exibidos em uma lista para visualização.

## Pontos críticos para teste

Validação de campos obrigatórios no cadastro de cursos.

Não há limite de caracteres

Sistema permite cadastrar cursos duplicados

Organização e alinhamento dos elementos da interface.

Exibição de imagens associadas aos cursos.

Exibição correta das informações cadastradas na listagem de cursos.

Funcionamento correto da funcionalidade de exclusão de cursos.
